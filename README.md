# Pet-Feeder
Our project is a smart pet feeder system built using the ESP32 microcontroller. It combines an LCD display, a real-time clock (RTC), a servo motor for dispensing food, and an RFID module for pet identification. The system is designed to automatically dispense food every 4 hours and allows pets to request food using their RFID tags. Each pet is assigned a unique RFID tag. When a pet approaches the feeder and scans its tag, the system checks the last feeding time. If the pet was fed more than 2 hours ago, the servo activates to dispense food; otherwise, the LCD displays a message indicating it is not yet time. All components communicate over a shared I2C bus, and the RFID module is connected on a separate SPI interface, using a dedicated breadboard for isolation The project ensures fair feeding schedules, prevents overfeeding, and demonstrates effective integration of sensors and actuators in an embedded IoT application.
